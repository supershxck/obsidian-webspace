In Obsidian, slash commands (introduced in later versions) allow you to quickly perform actions or insert elements into your notes by typing `/` followed by a command. Here are some common slash commands you can use:

### 1. **Formatting and Text**
   - `/heading` or `/h` - Insert a heading (e.g., H1, H2, H3).
   - `/bold` or `/b` - Insert bold text.
   - `/italic` or `/i` - Insert italic text.
   - `/highlight` - Insert highlighted text.
   - `/strikethrough` or `/s` - Insert strikethrough text.
   - `/code` - Insert inline code.
   - `/codeblock` - Insert a code block.
   - `/quote` or `/blockquote` - Insert a blockquote.
   - `/list` - Insert a bulleted or numbered list.
   - `/task` or `/todo` - Insert a task or checkbox.

### 2. **Media and Embeds**
   - `/image` - Insert an image.
   - `/link` - Insert a link.
   - `/embed` - Embed a file or note.
   - `/audio` - Insert an audio file.
   - `/video` - Insert a video file.

### 3. **Notes and Files**
   - `/note` - Create a link to another note.
   - `/tag` - Insert a tag.
   - `/template` - Insert a template.
   - `/file` - Insert a file or attachment.

### 4. **Advanced Features**
   - `/table` - Insert a table.
   - `/math` - Insert a math equation or block.
   - `/mermaid` - Insert a Mermaid diagram.
   - `/canvas` - Insert a canvas.
   - `/callout` - Insert a callout block.
   - `/dataview` - Insert a Dataview query (if Dataview plugin is installed).

### 5. **Custom Commands**
   - If you have custom plugins or scripts, they may also add their own slash commands.

### How to Use Slash Commands:
1. Open a note in Obsidian.
2. Type `/` to bring up the slash command menu.
3. Start typing the command name (e.g., `/table`).
4. Select the command from the dropdown or press `Enter` to execute it.

### Notes:
- The available slash commands may vary depending on the plugins you have installed.
- Some commands may require specific plugins (e.g., Dataview, Templates, etc.) to function.
